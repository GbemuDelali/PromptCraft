Example 15: Iterative Prompt Refinement

openprompt Copy code # Iterative Prompt Refinement Example template iterativeRefinementPrompt { "Generate a short story about a mysterious artifact. After each response, refine the prompt based on the generated content." } prompt(iterativeRefinementPrompt)

In this example, a template is used to iteratively refine a story prompt. After each response, the prompt is refined based on the generated content, allowing for an interactive storytelling experience.

Example 16: Contextual Parameterization

openprompt Copy code # Contextual Parameterization Example template contextualParameterPrompt { "Given the context of <>, provide insights into the impact of emerging technologies on <>." } prompt(contextualParameterPrompt, { "context": "current global events", "industry": "finance" })

This example showcases contextual parameterization, where the context and industry parameters are dynamically inserted into the prompt to generate insights specific to the given context.

Example 17: Multi-Step Inference

openprompt Copy code # Multi-Step Inference Example template multiStepInferencePrompt { "Based on the provided information, make a series of inferences about the potential outcomes and implications of a futuristic scenario involving <>." } prompt(multiStepInferencePrompt, { "elements": "advanced AI, sustainable energy, space exploration" })

This example involves multi-step inference, instructing GPT-3.5 to make a series of inferences about potential outcomes and implications of a futuristic scenario based on provided elements.

Example 18: Guided Creative Writing

openprompt Copy code # Guided Creative Writing Example template guidedWritingPrompt { "Start a creative writing piece with the given sentence: <>. Continue the story, introducing elements of suspense and unexpected twists." } prompt(guidedWritingPrompt, { "initialSentence": "In the quiet town of Everwood, a peculiar event unfolded on a stormy night." })

Here, the guided writing prompt instructs GPT-3.5 to start a creative writing piece with a provided sentence and continue the story by introducing elements of suspense and unexpected twists.

Example 19: User-Driven Story Generation

openprompt Copy code # User-Driven Story Generation Example template userDrivenStoryPrompt { "Let's co-create a story! You start by providing the opening sentence, and I'll build on it. Here's your starting point: <>." } prompt(userDrivenStoryPrompt, { "userOpeningSentence": "In a distant galaxy, a lone explorer embarked on a journey to unravel the mysteries of an undiscovered planet." })

This example demonstrates user-driven story generation, where the user provides the opening sentence, and GPT-3.5 collaboratively builds on it to co-create a story.

Example 20: Dynamic Scenario Mapping

openprompt Copy code # Dynamic Scenario Mapping Example template dynamicScenarioMappingPrompt { "Given the evolving landscape of technology and society, dynamically map the potential scenarios and challenges that may arise in the next <> years." } prompt(dynamicScenarioMappingPrompt, { "years": 10 })

Instruct GPT-3.5 to dynamically map potential scenarios and challenges that may arise in the next specified number of years, adapting to the evolving landscape of technology and society.
